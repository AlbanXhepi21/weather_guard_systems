# Product Catalogs

Place your PDF catalog files in this folder:

## Required Files

1. **olympos-guillotine-glass-systems.pdf** - Olympos Guillotine Glass System catalog from Kenan Metal
2. **bioclimatic-system-kenan-metal.pdf** - Bioclimatic Pergola Systems catalog from Kenan Metal

## File Naming

- Use lowercase letters
- Replace spaces with hyphens
- Keep file names URL-friendly

## Example

```
public/
└── catalogs/
    ├── olympos-guillotine-glass-systems.pdf
    └── bioclimatic-system-kenan-metal.pdf
```

## Notes

- Maximum recommended file size: 10MB per PDF
- Optimize PDFs for web viewing if possible
- Ensure PDFs are the latest version from Kenan Metal / Artminyum
